## 💎 ブルーアーカイブ

アビ夏新規なのですが、まともにプレイしていない期間がかなり長く、きちんとプレイし始めたのは戦術対抗戦シーズン7頃の時期（2024夏あたり）です。<br/>

---

最終編のような体験をもう一度味わいたいと思っていましたが、今となってはあまり期待せずにいます。現在は、ある先生の言葉を借りれば、ソシャゲのソシャの部分がゲームを続けている動機です。

---

ボスコンテンツは現在あまりモチベーションが高くありませんが、きちんとプレイをし始めてから数ヶ月はちゃんとやってました。一応、Lunatic も複数クリアしていますが、ほとんどこだわっていません。

- 最高順位: 207位 (総力戦 市街地 ヒエロニムス シーズン72)

---

サークルは現在、「課金よりおいしいご飯」に所属しています。

- プレイ開始: 非SNS系のサークル
- 2024年10月: 課金よりおいしいご飯
